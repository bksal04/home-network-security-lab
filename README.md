# home-network-security-lab
This project documents my home network setup using a Raspberry Pi 5, Pi-hole, and Tailscale to improve security, visibility, and remote access.

## My Setup
- Raspberry Pi 5 running Pi-hole and Tailscale
- Multiple home devices connected (laptop, phone)
- Pi-hole configured as network-wide DNS server
- Tailscale used for secure remote access without port forwarding
