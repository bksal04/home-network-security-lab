# Pi-hole Setup

## Host Platform
- Raspberry Pi 5
- Linux-based deployment

## Configuration Summary
- Installed Pi-hole on the Raspberry Pi
- Configured the device with a consistent local IP address
- Set Pi-hole as the primary DNS resolver for selected home devices
- Enabled blocklists for ads and suspicious domains

## Security Value
Pi-hole improves security by filtering DNS requests before devices connect to unwanted domains. This reduces exposure to malicious infrastructure and improves visibility into DNS activity.

## Validation
- Verified DNS queries were visible in the Pi-hole dashboard
- Confirmed blocked domains were being filtered correctly
- Tested multiple home devices against the resolver
